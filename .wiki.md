# Project Summary
Onkati is a neighborhood loyalty platform aimed at enhancing customer engagement and retention by allowing users to earn points at local merchants. By leveraging modern technologies like React and Supabase, it provides an intuitive user experience with features such as QR code scanning and transaction tracking. Recent updates include a complete redesign of the customer panel interface, enabling customers to request points using only a store code, and significant improvements to error handling.

# Project Module Description
The project consists of several functional modules:
- **Authentication**: Manages user registration, login, and session management.
- **Admin Panel**: Allows administrators to oversee users and transactions and manage merchant settings.
- **Customer Panel**: Enables customers to manage profiles, view transaction history, and earn/redeem points. Recently updated to allow point requests with only a store code.
- **Merchant Panel**: Empowers merchants to generate QR codes, manage store codes, and handle transactions. Redesigned to include tabs for managing requests, calendar, customers, and rate settings.
- **Dashboard**: Offers an overview of user and transaction data.
- **Supabase Functions**: Implements serverless functions for user registration and transaction handling.
- **Security Layer**: Incorporates security measures to protect user data.

# Technology Stack
- **Frontend**: React, TypeScript, Tailwind CSS
- **Backend**: Supabase (PostgreSQL)
- **Build Tool**: Vite

# Usage
1. **Install Dependencies**: Run `npm install` to install all project dependencies.
2. **Build the Project**: Use `npm run build` to build the project for production.
3. **Run the Project**: Use `npm run dev` to start the development environment.
