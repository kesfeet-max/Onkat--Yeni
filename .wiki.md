# Project Summary
Onkati is a neighborhood loyalty platform designed to enhance customer engagement and retention by allowing users to earn points at local merchants. It leverages modern technologies like React and Supabase to provide a seamless user experience, incorporating features such as QR code scanning and transaction tracking. Recent updates have focused on optimizing the project for production deployment, cleaning up the directory structure, and ensuring compatibility with deployment environments.

# Project Module Description
The project consists of several functional modules:
- **Authentication**: Manages user registration, login, and session management.
- **Admin Panel**: Allows administrators to oversee users and transactions and manage merchant settings.
- **Customer Panel**: Enables customers to manage profiles, view transaction history, and earn/redeem points.
- **Merchant Panel**: Empowers merchants to generate QR codes, manage store codes, and handle transactions.
- **Dashboard**: Offers an overview of user and transaction data.
- **Supabase Functions**: Implements serverless functions for user registration and transaction handling.
- **Security Layer**: Incorporates security measures to protect user data.

# File Description Inventory
- **.npmrc**: Configuration to prevent dependency conflicts.
- **index.html**: Main entry point for the application.
- **package.json**: Contains project metadata and essential dependencies.
- **package-lock.json**: Ensures consistent installations across environments.
- **vite.config.ts**: Configuration for the Vite build tool.
- **src/App.tsx**: Main application component.
- **src/main.tsx**: Renders the application.
- **src/lib/**: Contains utility functions and libraries.
- **src/pages/**: Contains different application pages including login, registration, and admin panels.
- **src/components/**: Contains reusable UI components.
- **public/_redirects**: Configuration for SPA routing.
- **render.yaml**: Configuration for deployment settings.

# Technology Stack
- **Frontend**: React, TypeScript, Tailwind CSS
- **Backend**: Supabase (PostgreSQL)
- **Build Tool**: Vite

# Usage
1. **Install Dependencies**: Run `npm install` to install all project dependencies.
2. **Build the Project**: Use `npm run build` to build the project for production.
3. **Run the Project**: Use `npm run dev` to start the development environment.
